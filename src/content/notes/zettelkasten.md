# 卡片笔记法实践

## 什么是卡片笔记法？

卡片笔记法（Zettelkasten）是德国社会学家尼克拉斯·卢曼发明的一种知识管理方法。卢曼一生出版了超过70本书籍和400篇学术文章，很大程度上归功于这个方法。

> "我不思考，我只是把笔记放进盒子里。" —— 尼克拉斯·卢曼

## 核心原则

### 1. 原子化

每张笔记只包含**一个想法**：

- ✅ 一个概念、一个观点、一个事实
- ❌ 不要把多个主题混在一起

### 2. 用自己的话写

不是简单的摘抄，而是：

- 理解后的重新表达
- 加入自己的思考和见解
- 建立与已有知识的联系

### 3. 建立连接

笔记之间通过链接形成网络：

```
[[学习笔记]] → [[卡片笔记法]]
     ↓
[[知识管理工具]]
     ↓
[[Obsidian使用技巧]]
```

## 我的实践

### 工具选择

目前使用 **Obsidian** 作为主要工具：

- 本地存储，数据安全
- 双向链接
- 图谱视图
- 丰富的插件生态

### 工作流程

```
阅读/学习 → 闪念笔记 → 永久笔记 → 索引笔记
    ↓           ↓           ↓
  高亮       整理思考     建立连接
```

### 每日习惯

1. **晨间回顾**：浏览昨天的笔记，建立连接
2. **随时记录**：使用手机快速捕捉想法
3. **晚间整理**：将闪念笔记转化为永久笔记

## 收获与反思

坚持使用卡片笔记法一年后：

- 知识不再孤立，而是形成网络
- 写作变得更容易，素材信手拈来
- 思考更加深入，能看到事物间的联系
